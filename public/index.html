Checkpoint 1: CRUD App

Overview

This app manages recipes for a website. You can create, review, update, and delete recipes.

Routes

  • GET /recipes: Lists the entire array of recipes from the data/index.
  • GET /recipes/:id: Shows the recipes that matches the passed-in id.
  • POST /recipes: Creates a new recipe and adds it to the current array of recipes.
  • PUT /recipes/:id: Updates one recipe in the array based on the passed-in id.
  • DELETE /recipes/:id: Deletes one recipe in the array based on its id
